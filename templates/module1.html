Listas enlazadas



Concepto


Las listas son un tipo de estructura lineal y dinámica de datos. Estas estan compuestas por nodos los cuales tiene 2 partes, un indicador el cual guarda la direccion de memoria del nodo que va proximo a este y el elemento que pertenece a la lista.
Las listas pueden ser

  • Simplemente enlazadas, cuando solo se pueden recorrer en un solo sentido, es decir sus nodos incluyen una referencia al nodo que les sigue, pero no al anterior
  • Doblemente enlazada, cuando sus nodos contienen referencias al nodo anterior y al proximo haciendo que se pueda recorrer hacia cualquier sentido.
  • Circulares cuando el ultimo objeto, llamado cola, tiene una referencia al primer nodo, llamado cabeza, como si fuera su siguiente


    • Operaciones permitidas


  • Añadir, sea en la cabeza o en la cola
  • Eliminar por nodo o por objeto
  • Ordenar
  • Buscar
  • Limpiar la lista
  • Crear una sublista

  • Ventajas y desventajas


  • Insercion y extraccion de nodos sencilla
  • Concatenacion y particion sencilla
  • El uso de memoria se adapta a la cantidad de datos almacenados
  • Dificultad para acceder a posiciones intermedias


    • Pilas



    Concepto


    Las pilas son una estructura de datos no fundamental, la cual requiere de, ya sea una lista enlazada o un arreglo para poder realizarse o presentarse. Esta estructura se caracteriza con el principio de Li-Fo, el cual significa Last in, First Out, representando que el ultimo elemento añadido es el primero al que se puede acceder en la pila.

    Operaciones permitidas


  • Añadir a la pila
  • Eliminar el ultimo elemento añadido
  • Ordenar
  • Buscar

    • Ventajas y desventajas


    Colas



    Concepto


    Las colas, asi como las pilas son una estructura de datos no fundamental que requiere la utilizacion de una lista o un arreglo ser implementada. A diferencia de las pilas, las colas se rigen bajo el principio de Fi-Fo, refiriendose a First in, First out, refiriendose a que el primer elemento que sea introducido, sera el primero al que se puede acceder externamente.
    Otra variante de estas son las colas con prioridad, las cuales manejan un nivel de prioridad para los elementos que son introducidos a la misma, donde los elementos que tengan la mayor prioridad, se posicionaran al inicio de la cola. Si hay elementos con el mismo nivel de prioridad, cumpliran con el Fi-Fo, ordenandose por orden de insercion.
