En este modulo puedes ingresar codigo fuente tuyo de Python para ver una representacion grafica del mismo.

El circulo rojo define el inicio de un script en python, es la representacion de un modulo.

El circulo azul representa la declaracion de una clase.

El cuadrado morado representa la declaracion de una funcion.

El triangulo verde representa los argumentos de una funcion.

El rombo amarillo representa los atributos de una clase.
